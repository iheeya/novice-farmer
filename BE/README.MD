## 2. 서비스 개요
 - 베란다, 주말농장 등 주변에서 작물을 간단하게 키우고 싶은 도시농부들이 텃밭과 작물을 추천받고, 키우는 작물을 관리할 수 있는 서비스

 ### 2) 텃밭/작물 추천 및 등록

 ### 5) 태그 기반 커뮤니티
